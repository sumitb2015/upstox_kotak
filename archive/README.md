# Archive Directory

This directory contains archived files that are not part of the active codebase but are kept for reference.

## Structure

### `debug_scripts/`
Debug and testing scripts used during development:
- `debug_*.py` - Various debugging utilities for testing API, prices, Supertrend, etc.
- `check_*.py` - SDK and signal checking utilities
- `verify_refactor.py` - Refactoring verification script
- `robust_fix.py` - Historical fix implementation
- `temp_renko_ema.py` - Temporary Renko EMA testing
- `test_websocket_stream.py` - WebSocket streaming tests
- `find_vix.py` - VIX data finder
- `backtest.py` - Backtesting utility

### `temp_files/`
Temporary files from development:
- `CODE_REVIEW_COMPLETE.md` - Historical code review documentation
- `auth_error_page.html` - Authentication error page template

### `old_logs/`
Archived log files and state dumps:
- `*.log` - Strategy execution logs
- `*.json` - State snapshots
- `*.csv` - Historical reports

## Note
These files are kept for historical reference and debugging purposes. They are not part of the active trading system.
